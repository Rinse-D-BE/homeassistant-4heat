# 4Heat Stove
**Integration domain:** `fourheat`
This integration uses the domain `fourheat` inside Home Assistant.

Home Assistant integration for 4Heat WiFi stoves.


## Features
- Local polling
- Sensors for stove status
- Switches for control
- Config flow (UI setup)

## Installation (HACS)
1. Open HACS
2. Integrations
3. Add custom repository:
   - URL: https://github.com/zaubererty/homeassistant-4heat
   - Category: Integration
4. Install "4Heat Stove"
5. Restart Home Assistant

## Configuration
Configuration is done via the Home Assistant UI.

## Supported versions
- Home Assistant Core 2026.2.0 or newer
